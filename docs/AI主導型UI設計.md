# AI主導型UI設計

## 概要
補助金申請書作成を**AI主導**で進める対話型UIの設計です。

**コンセプト:**
- ChatGPT/Claude風の対話形式
- AIが店舗情報を把握してから質問を開始
- AIが申請書の内容を想定しながらユーザーを誘導
- ユーザーは確実に答えられることだけを回答

---

## 🎯 基本設計思想

### 【従来型（質問リスト方式）】
```
システム: Q1を表示
ユーザー: 回答
システム: Q2を表示
ユーザー: 回答
...
```
→ 機械的で冷たい印象

### 【AI主導型（対話方式）】
```
AI: 「まず、お店の名前を教えてください」
ユーザー: 「〇〇レストラン」
AI: 「Google Mapsで見つけました！（地図・写真表示）
     フレンチレストランなんですね。素敵です✨
     口コミも★4.6と高評価ですね。」
AI: 「では、〇〇さん（代表者名）にいくつか質問させてください。
     開業されたのはいつ頃ですか？」
ユーザー: 「2018年です」
AI: 「なるほど、創業6年目なんですね。
     では、最近の月間売上はどのくらいですか？」
```
→ 自然な会話形式で進行

## 🎯 AIの質問戦略

### 【1. 確実に答えられることから聞く】

❌ 「直近1期の売上総利益は？」
✅ 「最近の月間売上はどのくらいですか？」

### 【2. 答えられない質問はスキップ】

```
AI: 「営業利益率はどのくらいですか？」

[選択肢]
• 10%未満
• 10～20%
• 20%以上
• わかりません

→「わかりません」を選択

AI: 「承知しました。
では、一般的な飲食業の利益率（10～15%）
で仮計算しておきますね。

後で正確な数字がわかったら、
修正できます。」
```

### 【3. Google Maps情報を最大活用】

```
AI: 「口コミを分析したところ、
以下の評価が多いです：

• 料理が美味しい（52件）
• 雰囲気が良い（38件）
• 記念日にぴったり（25件）

これらを「強み」として
申請書に記載しますね！」
```

### 【4. AIが推定した内容を確認】

```
AI: 「フレンチレストランの場合、
一般的に以下のような課題があります：

⚠️ 原材料費の高騰（特にバター・小麦粉）
⚠️ 光熱費（ガス・電気）の上昇

山田さんのお店も、
この影響を受けていますか？」

[はい、影響を受けています]
[いいえ、あまり影響はありません]
```

## 🔄 対話の自然さを高める工夫

### 【1. 相槌・共感を入れる】

```
ユーザー: 「新規顧客が増えない」
AI: 「そうなんですね。
口コミ評価が高いのに、
新規顧客が増えないのはもどかしいですよね。」
```

### 【2. 前の回答を踏まえた質問】

```
ユーザー: 「月間売上200万円」
AI: 「年間だと約2,400万円ですね。
では、そのうち利益はどのくらいですか？」
```

### 【3. 理由・背景を説明】

```
AI: 「補助金の審査では、
【現状】→【課題】→【解決策】→【効果】
という流れが重要です。

なので、現在の課題について
詳しく教えてください。」
```

### 【4. 進捗を見せる】

```
━━━━━━━━━━━━━━━
進捗: ■■■■■□□□□□ 50%
推定残り時間: 約10分
━━━━━━━━━━━━━━━

AI: 「もう半分まで来ました！
あと少しで完成です💪」
```

## 📱 実装イメージ（React Component）

### 【基本構造】

```jsx
<ChatContainer>
  {/* メッセージ履歴 */}
  <MessageList>
    <AIMessage>
      まず、お店の名前を教えてください
    </AIMessage>

    <UserMessage>
      〇〇レストラン
    </UserMessage>

    <AIMessage>
      <GoogleMapsCard />
      見つかりました！
      フレンチレストランなんですね✨
    </AIMessage>

    <AIMessage type="question">
      代表者のお名前を教えてください
      <TextInput />
    </AIMessage>
  </MessageList>

  {/* 進捗バー */}
  <ProgressBar value={30} />
</ChatContainer>
```

### 【メッセージタイプ】

1. **AIメッセージ（通常）**
   - テキストのみ
   - 相槌・共感

2. **AIメッセージ（質問）**
   - テキスト + 入力欄
   - 選択肢ボタン

3. **AIメッセージ（情報表示）**
   - Google Mapsカード
   - 分析結果の表示
   - 申請書プレビュー

4. **ユーザーメッセージ**
   - 回答内容

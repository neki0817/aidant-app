# 様式2作成のための質問フロー設計

## 概要
小規模事業者持続化補助金の申請書類「経営計画書兼補助事業計画書（様式2）」を作成するための質問フローです。

**設計原則:**
- **一問一答式**: 1つの質問で1つの回答のみを求める
- **段階的な深掘り**: 回答に応じて次の質問を出し分ける
- **AI補完前提**: 最小限の質問で、AIが残りを推定・補完する

---

## 📋 質問フロー全体像

```
【Phase 1: 基本情報収集】(Q1-Q10)
 ↓
【Phase 2: 売上・業績情報】(Q11-Q20)
 ↓
【Phase 3: 商品・サービス詳細】(Q21-Q30)
 ↓
【Phase 4: 顧客・市場情報】(Q31-Q40)
 ↓
【Phase 5: 課題・目標設定】(Q41-Q50)
 ↓
【Phase 6: 補助事業計画】(Q51-Q60)
```

## 🎯 質問フロー設計のポイント

### 【一問一答の徹底】
- 1つの質問では1つの情報のみを収集
- 複数の情報が必要な場合は、質問を分割
- 条件分岐を活用して、必要な質問のみを表示

### 【段階的な深掘り】
```
基本情報 → 詳細情報 → 補足情報
```
という流れで、徐々に詳しく聞いていく

### 【条件分岐の活用】
- 前の回答によって次の質問を出し分ける
- 不要な質問はスキップする
- ユーザーの回答負担を最小化

### 【AI補完の活用】
- Google Maps情報を活用
- 業界標準データで補完
- ユーザーが「わからない」場合はAI推定

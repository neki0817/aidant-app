## 1. 様式2の構成パターン（飲食業）

### 1-1. 企業概要

**記載すべき情報:**
- 屋号、代表者、創業年月日、所在地
- コンセプト（例: 和風モダンカフェ）
- 提供商品（料理、ドリンク、スイーツ）
- 看板メニュー（例: 月替わり御膳 1,500円）
- 従業員数（正社員、パート・アルバイト）
- 営業時間帯（ランチ、カフェ、ディナー）

**実例（サンプル庵）:**
```
和風モダンカフェ「サンプル庵」は、日本の「和」と現代的な「モダン」を融合させた空間で、
厳選された九州産食材の創作和風料理、日本茶・珈琲、季節の和スイーツを提供するカフェである。
ランチ、カフェ、ディナーの各時間帯で運営し、特に地元の契約農家や漁港直送の食材を使用した
「月替わり御膳」が人気である。
```

**AI質問生成への活用:**
- Q: 「お店のコンセプトを一言で教えてください」
- 回答例: 「和風モダン」「カジュアルフレンチ」「地産地消イタリアン」
- Q: 「看板メニューは何ですか? 価格も教えてください」
- 回答例: 「ハンバーグランチ 1,200円」

---

### 1-2. 売上・利益の状況

**記載すべき情報（表形式）:**

| 時間帯 | 売上高 | 売上総利益 | 売上シェア | 客単価 |
|-------|--------|----------|----------|--------|
| ランチタイム | 4,500千円 | 2,700千円 | 50.0% | 約1,200円 |
| カフェタイム | 2,700千円 | 1,620千円 | 30.0% | 約800円 |
| ディナータイム | 1,800千円 | 1,080千円 | 20.0% | 約2,500円 |
| 合計 | 9,000千円 | 5,400千円 | 100.0% | 約1,300円 |

**AI質問生成への活用:**
- Q: 「月間の平均売上はいくらですか?」
- Q: 「ランチ、カフェ、ディナーのうち、どの時間帯の売上が多いですか?」
- Q: 「客単価はいくらぐらいですか?」

**計算ロジック:**
- 売上総利益率 = 売上総利益 / 売上高 = 5,400 / 9,000 = 60%
- 売上シェア = 各時間帯の売上 / 合計売上

### 1-3. 経営課題

**典型的な課題（飲食業）:**

1. **平日のランチタイム以降の集客力と客単価の向上**
   - カフェ・ディナータイムの稼働率が低い
   - 明確な来店動機が不足

2. **メニュー開発とプロモーションの強化**
   - 季節限定メニューのバリエーション不足
   - デジタル媒体を活用した効果的な情報発信が不足

3. **調理・提供プロセスの効率化**
   - 手作業に頼る業務が多い
   - ピーク時の提供遅延や人件費率の圧迫

**AI質問生成への活用:**
- Q: 「今、一番困っていることは何ですか?」
- 選択肢:
  - 「新規顧客が増えない」
  - 「リピーターが少ない」
  - 「平日の昼以外の売上が少ない」
  - 「人手不足」
  - 「原材料費が高い」

## 2. 顧客ニーズと市場の動向

### 2-1. 市場の動向（飲食業で使えるデータ）

**インバウンド需要（福岡の例）:**
```
福岡市は、国際線就航便数の回復と地理的優位性から、インバウンド需要が急速に高まっている。
福岡空港の国際線旅客数は2024年3月時点でコロナ禍前の約90%まで回復しており、
特に韓国・中国・台湾からの観光客が顕著である。
```

**健康志向と体験型消費ニーズの増加:**
```
2024年の○○社の消費者調査では、「食材の産地や安全性にこだわる」と回答した消費者が
全体の70%に達している。また、「モノ消費」から「コト消費」への移行が進み、
「体験型消費」への関心が増加している。
```

**AI質問生成への活用:**
- Q: 「最近、お客様の傾向で変化を感じることはありますか?」
- 回答例提示: 「健康志向の方が増えた」「インスタ映えを意識する方が増えた」

### 2-2. 顧客ニーズ

**当店が選ばれる理由（サンプル庵の例）:**

Google口コミや店内アンケートから抽出された評価:
1. 「落ち着いた和モダンな空間でリラックスできる」
2. 「九州産食材の料理が美味しい」
3. 「抹茶スイーツが本格的でSNS映えする」

**改善点:**
1. 「外国人観光客への対応が不十分」
2. 「予約が取りにくい時がある」
3. 「キャッシュレス決済の種類を増やしてほしい」

**AI質問生成への活用:**
- Q: 「Google Mapsの口コミを分析したところ、『雰囲気が良い』というコメントが多いですが、これは意識していますか?」
- Q: 「お客様から改善してほしいと言われたことはありますか?」

## 3. 自社の強みと弱み

### 3-1. 強み（飲食業の典型例）

**① 独自のコンセプトと空間デザイン**
```
「和」と「モダン」を融合させた唯一無二の空間は、顧客から高い評価を得ている。
Google口コミでは「洗練された空間で心が安らぐ」「写真映えするところが多くて楽しい」
といった声が多数寄せられている。来店客の約○%がSNSに投稿するなど、
そのデザイン性が集客に貢献している。
```

**② 厳選された食材へのこだわり**
```
安心・安全で高品質な料理を提供するため、食材には徹底してこだわっている。
肉類は「糸島産○○鳥」、野菜は「○○市契約農家 ○○農園」から直接仕入れ、
魚介類は○○鮮魚市場から毎日直送されるなど、生産者の顔が見える食材を使用している。
```

**③ 創作和風料理と専門性**
```
代表の○○が「日本茶インストラクター」の資格を持ち、全国各地の厳選された日本茶を
独自の淹れ方で提供している点は、他にはない強みである。
```

**④ アクセスの良い立地**
```
福岡市天神の中心地という交通の便が良い立地は、高い集客ポテンシャルを有する。
西鉄福岡（天神）駅から徒歩〇分、地下鉄天神駅から徒歩〇分という利便性は、
ビジネス層から観光客まで幅広い層の来店を促進している。
```

**AI質問生成への活用:**
- Q: 「お客様から『これが良い』と評価されているのはどんな点ですか?」
- 選択肢: 「味」「雰囲気」「接客」「価格」「立地」「その他」
- Q: 「食材の仕入れで特にこだわっている点はありますか?」
- 回答例提示: 「地元の契約農家から仕入れ」「無農薬野菜」「産地直送」

---

### 3-2. 弱み（飲食業の典型例）

**① 認知度の低さと販路開拓の遅れ**
```
創業間もないため、地域や潜在顧客層への認知度が不足しており、顧客開拓が遅れている。
特に、ターゲットと定めている20～30代のデジタルネイティブ層へのリーチが不十分である。
```

**② プロモーション・情報発信の不足**
```
新メニューやイベントの告知が主に店舗内で行われ、デジタルマーケティングが手薄である。
SNSの更新頻度が低く、ターゲット層に響く魅力的なコンテンツの発信ができていないことが課題である。
```

**③ 調理・提供プロセスの非効率性**
```
手作業による工程が多く、ピーク時の提供遅延や人件費負担増がある。
特にランチタイムの「月替わり御膳」の調理には手間がかかり、
ピーク時に提供まで15分以上かかることもあり、機会損失に繋がっている。
```

**④ 外国人観光客への対応不足**
```
多言語メニューや外国語での接客、効果的なプロモーションが不十分である。
特に、主要なインバウンド層である韓国語・中国語での対応ができていないため、
来店機会を逃している。キャッシュレス決済の種類も限定的である。
```

**AI質問生成への活用:**
- Q: 「今、改善したいと思っていることは何ですか?」
- 選択肢: 「認知度を上げたい」「SNSでの発信を増やしたい」「調理を効率化したい」「外国人対応を強化したい」

## 6. AI質問生成への活用方法

### 6-1. Phase 2（顧客ニーズと市場の動向）の質問パターン

**質問パターン1: ターゲット顧客の確認**
```
【AI】「お客様の年齢層はどのあたりが多いですか?」
回答例: 「20代」「30-40代」「50代以上」「幅広い」

【ユーザー】「30-40代」

【AI】「30-40代のお客様が多いのですね。男女比はどうですか?」
回答例: 「女性が多い」「男性が多い」「半々」
```

**質問パターン2: 選ばれる理由**
```
【AI】「お客様が御店を選ぶ理由は何だと思いますか?」
選択肢:
□ 味が美味しい
□ 価格が手頃
□ 雰囲気が良い
□ 立地が便利
□ 接客が良い
□ その他

【ユーザー】「味が美味しい、雰囲気が良い」を選択

【AI】「味と雰囲気が強みなのですね。Google Mapsの口コミでも
     『料理が美味しい』というコメントが5件ありますね。
     特に評価されているメニューは何ですか?」

【ユーザー】「ハンバーグです」
```

### 6-2. Phase 3（自社の強み）の質問パターン

**質問パターン1: 強みの確認**
```
【AI】「お客様から特に評価されているのはどの点ですか?」
選択肢:
□ 味の良さ
□ ボリューム
□ 価格
□ 接客
□ 雰囲気
□ その他
```

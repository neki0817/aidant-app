# 顧客ニーズと市場の動向 記載ガイドライン

## 概要
小規模事業者持続化補助金の申請書類「経営計画書兼補助事業計画書(様式2)」における**2.顧客ニーズと市場の動向**の記載方法をまとめたガイドラインです。

---

## 📋 記載すべき項目一覧

### 【1. 業界市場全体について】

#### ■ 市場規模・動向
- **業界全体の市場規模**（できれば出典を明記）
- **成長/縮小傾向**
- **コロナ禍前後の変化**
- **今後の見通し**

**記載例:**
```
日本フードサービス協会の調査によると、国内の洋風フルサービス
レストラン市場規模は、2019年をピークに減少傾向にありましたが、
2023年にはコロナ前の約85%まで回復しています。
（出典：日本フードサービス協会「外食産業市場動向調査」2023年版）

特にフレンチレストランにおいては、以下の傾向が見られます：
- 高単価業態は回復が遅れている（コロナ前の70%程度）
- カジュアルフレンチは堅調（コロナ前水準を回復）
- ランチ需要は完全回復（2019年比105%）
```

#### ■ 消費者の嗜好変化
- **購買行動の変化**
- **価値観の変化**
- **新しいトレンド**

**記載例:**
```
コロナ禍を経て、消費者ニーズに以下の変化が見られます：

- "日常使い"できるフレンチへのニーズ増加
  「特別な日だけでなく、気軽に本格的な料理を楽しみたい」
  という層が拡大（20～40代女性を中心に）

- コストパフォーマンス重視
  「高品質な料理を適正価格で」という価値観が浸透

- 地産地消・サステナビリティへの関心
  地元食材を使用した料理や、フードロス削減への
  取り組みが支持を集めている
```

#### ■ 競合状況
- **地域内の競合店舗数**
- **競合の業態・価格帯**
- **市場シェア**

**記載例:**
```
当店が位置する○○市内では、以下の競合状況となっています：

- 高級フレンチ: 2店舗（客単価15,000円以上）
  → 廃業やカジュアル業態への転換で減少傾向

- カジュアルフレンチ: 5店舗（客単価5,000～10,000円）
  → 当店を含め、堅調に推移

- イタリアン: 8店舗
  → フレンチよりも参入障壁が低く、競合として無視できない存在
```

### 【2. 自社事業に関わる市場】

#### ■ 商圏分析（地域別顧客データ）

地域別の売上構成・認知率・開拓余地を表形式で整理:

```
| 地域 | 顧客数 | 売上構成比 | 人口 | 人口比顧客率 | 開拓余地 |
|------|--------|-----------|------|------------|---------|
| ○○市 | 1,800名 | 72% | 120,000人 | 1.5% | 中 |
| A市   | 450名  | 18% | 80,000人  | 0.56% | 大 |
| B町   | 150名  | 6%  | 30,000人  | 0.5%  | 大 |
| その他 | 100名  | 4%  | -        | -     | 大 |
```

**分析結果の記載:**
```
- ○○市（地元）: 当店の主要商圏
  - 売上の7割以上を占める
  - 人口比1.5%の認知率は、同規模フレンチとして標準的
  - 駅から徒歩圏内の認知率は高いが、郊外エリアへの浸透が課題

- 隣接A市: 最大の成長余地
  - 人口8万人と大きいが、認知率は0.56%と低い
  - 車で15分圏内にもかかわらず、来店客は全体の18%に留まる
  - 広告宣伝の強化で顧客数の大幅増が見込める
```

#### ■ 購入商品の特徴

**サービス業（飲食・サロン等）の場合:**

メニュー別の売上構成・客層・特徴を表形式で整理:

```
| メニュー | 売上構成比 | 客層 | 特徴 |
|---------|----------|------|------|
| Bコース | 50% | 30～50代女性 | 平日の女性ランチ会で人気 |
| Aコース | 35% | 全年代 | ビジネスランチ・一人客 |
```

**季節変動の記載:**
```
- 春(3～5月)・秋(9～11月): 高単価コースの比率が上昇
  （記念日・歓送迎会需要）
- 夏(6～8月): 軽めのメニューが好まれ、客単価やや低下
- 冬(12～2月): 忘年会・新年会需要で高単価コースの比率上昇
```

**小売業の場合:**

商品カテゴリー別の売上構成・購買層・季節変動を記載:

```
| 商品カテゴリー | 売上構成比 | 主要購買層 | 季節変動 |
|--------------|----------|-----------|---------|
| レディース   | 60% | 30～50代女性 | 春秋が繁忙期 |
| メンズ       | 25% | 30～40代男性 | 冬物が好調 |
| アクセサリー | 15% | 20～40代女性 | 12月が最盛期 |
```

### 【3. 顧客ニーズの把握】

#### ■ 顧客アンケート結果

**実施した場合:**

```
【顧客アンケート結果】（2023年9～10月実施、回答数：120名）

■来店理由（複数回答可）
| 理由 | 回答率 |
|------|--------|
| 記念日・お祝い | 58% |
| 料理の質が高い | 52% |
| 雰囲気が良い | 45% |
| SNSで見て興味を持った | 32% |

■評価が高い点
| 項目 | 満足度 |
|------|--------|
| 料理の味 | 95% |
| 盛り付け・見た目 | 92% |
| スタッフの接客 | 88% |

■改善要望
| 項目 | 回答数 |
|------|--------|
| 駐車場が少ない | 38名 |
| 予約が取りにくい | 25名 |
```

**実施していない場合（AI推定）:**

Google Maps口コミ、SNS投稿、業界一般的な傾向から推定:

```
【顧客ニーズ（推定）】

■来店理由
業種・立地から推定すると、以下の理由での来店が多いと考えられます：
- 記念日・お祝い（飲食業の場合、50～60%程度）
- 料理の質・味への期待
- 友人・知人からの紹介

■評価されている点
Google Mapsの口コミ（★4.6 / 85件）から、以下の点が
高く評価されています：
- 料理の味・品質
- 店内の雰囲気
- スタッフの対応
```

#### ■ SNS・口コミサイトでの評価

**記載内容:**
- プラットフォーム名（Instagram、Google Maps、食べログ等）
- フォロワー数 / レビュー数 / 評価点数
- 高評価のポイント
- 改善要望

**記載例:**
```
【SNS・口コミサイトでの評価】

■Instagram（フォロワー: 1,200名）
- 「盛り付けが美しい」「SNS映えする」といったコメントが多数
- 料理写真の投稿が拡散され、新規顧客獲得につながっている

■Googleマイビジネス（★4.6 / 5.0、レビュー数：85件）
高評価のポイント:
- 「本格的なフレンチが手頃な価格で楽しめる」
- 「記念日にぴったり。サプライズ対応も丁寧」

改善要望:
- 「駐車場が遠い」
- 「人気で予約が取りづらい」
```

### 【4. 経営環境への影響】

#### ■ ウクライナ情勢・物価高騰の影響

**必須記載事項:**
- 影響を受けている費目
- 具体的な上昇率・金額
- 経営への影響
- 対応策
- 顧客の反応

**記載例:**

```
【原材料費の高騰】

2022年2月のウクライナ侵攻以降、以下の食材・資材で
大幅な価格上昇が発生しています：

| 品目 | 2022年2月 | 2023年12月 | 上昇率 |
|------|----------|-----------|--------|
| 小麦粉 | 200円/kg | 280円/kg | +40% |
| バター | 1,200円/kg | 1,680円/kg | +40% |
| 牛肉 | 3,500円/kg | 4,200円/kg | +20% |
| ガス・電気代 | 月35,000円 | 月52,000円 | +48.5% |

【当店への影響】
原材料費・光熱費の高騰により、原価率が従来の40%から45%に上昇。
売上が前年並みでも、営業利益が圧迫される状況となっています。

【対応策】
- メニュー価格の改定（2023年4月に全メニューを平均8%値上げ）
- 地元食材への切り替え（輸入食材の一部を地元産に変更）
- メニュー構成の見直し（原価率の高いメニューの頻度調整）

【顧客の反応】
価格改定については、「やむを得ない」との理解を示す顧客が多数。
ただし、さらなる値上げは客離れのリスクがあるため、
コスト削減と付加価値向上の両立が急務です。
```

## 🤖 AI自律エージェントによる補完方式

### 【基本方針】

ユーザーからの最小限の回答 + 外部データ（Google Maps、業界統計、SNS等）を組み合わせて、「顧客ニーズと市場の動向」を自動生成する。

### 【データソース】

#### 1. **ユーザー入力データ**
現在の質問フロー（Step1）から取得:
- 業種
- 開業日
- 月間売上
- 営業日数
- 従業員数
- 主要商品・サービス

#### 2. **Google Maps API**
- 口コミ（reviews）
  - 評価点数
  - レビュー内容（高評価・低評価の理由）
  - 投稿者の属性
- 営業時間
- 写真（料理・店内の様子）
- 混雑状況

#### 3. **業界統計データ（推定）**
業種ごとの一般的な市場動向:
- 飲食業: 日本フードサービス協会のデータ
- 小売業: 経済産業省の商業動態統計
- サービス業: 業界団体の統計データ

#### 4. **地域データ**
- 人口統計（総務省統計局）
- 商圏人口
- 競合店舗数（Google Maps検索）

## 📝 AI生成ロジックの設計

### 【生成フロー】

```
1. 業種判定
   ↓
2. 業界統計データの取得（または推定）
   ↓
3. Google Maps口コミ分析
   ↓
4. 商圏分析（地域人口データ + Google Maps）
   ↓
5. 競合分析（Google Maps検索）
   ↓
6. 物価高騰影響の推定（業種別の標準データ）
   ↓
7. 文章生成
```

### 【業種別生成パターン】

#### ■ 飲食業
- **市場動向**: 日本フードサービス協会データ（またはAI知識ベース）
- **顧客ニーズ**: Google Maps口コミから抽出
  - 「美味しい」「雰囲気が良い」等の頻出ワード分析
  - 評価点数の分布
- **商圏分析**: 店舗住所から半径5km圏内の人口
- **競合分析**: 同業種の店舗数（Google Maps検索）
- **物価高騰**: 食材費・光熱費の上昇率（業界平均データ）

#### ■ 小売業
- **市場動向**: 経済産業省の商業動態統計
- **顧客ニーズ**: Google Maps口コミ + 店舗写真分析
- **商圏分析**: 店舗住所から半径3km圏内の人口
- **競合分析**: 同業種の店舗数
- **物価高騰**: 仕入れコスト・光熱費の上昇

#### ■ サービス業（サロン等）
- **市場動向**: 業界団体データ（または一般的な傾向）
- **顧客ニーズ**: Google Maps口コミ + SNS投稿分析
- **商圏分析**: 店舗住所から半径10km圏内の人口
- **競合分析**: 同業種の店舗数
- **物価高騰**: 材料費・光熱費の上昇

## 🎯 AI生成時の注意点

### 【1. データの信頼性】

- **出典を明記する**:
  ```
  （出典：日本フードサービス協会「外食産業市場動向調査」2023年版）
  ```

- **推定データには注釈を付ける**:
  ```
  ※業界平均データから推定
  ※Google Maps口コミ（85件）の分析結果
  ```

### 【2. 業種特性の反映】

業種ごとに重要な指標が異なる:

- **飲食業**: 客単価・回転率・リピーター率
- **小売業**: 客単価・購買頻度・在庫回転率
- **サービス業**: 施術時間・予約稼働率・リピーター率

### 【3. 地域特性の考慮】

- 都市部 vs 地方
- 商業地 vs 住宅街
- 駅前 vs 郊外

### 【4. 具体性の確保】

抽象的な表現を避け、数値で示す:

❌ 悪い例: 「顧客満足度は高い」
✅ 良い例: 「Google Mapsの評価は★4.6/5.0（85件のレビュー）」

### 【5. 文章の自然さ】

- 機械的な文章にならないよう、接続詞・段落構成に配慮
- 記載例のトーンを参考に、審査員が読みやすい文章を生成

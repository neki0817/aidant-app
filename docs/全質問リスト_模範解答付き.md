### Q1-20: 売上総利益率（粗利率） ✨ NEW
**質問:**
売上総利益率（粗利率）はどのくらいですか？わからない場合は、業種の平均値を使用します

**回答例:**
```
65
```
（= 粗利率65%）

**模範解答（イタリアンレストラン）:**
```
65
```
（売上1,440万円 − 原価504万円 = 粗利936万円、粗利率65%）

**模範解答（カフェ）:**
```
70
```
（コーヒー・スイーツは原価率が低く、粗利率70%）

**模範解答（小売業）:**
```
35
```
（仕入れ販売のため粗利率は低め）

**回答例（わからない場合 - 業種平均を使用）:**
```
60
```
（= 飲食業の平均60%を使用）

**ヒント:**
- **粗利率 = (売上 − 原価) ÷ 売上 × 100**
- 飲食業：平均60%（原価率40%）
- 小売業：平均30%（原価率70%）
- サービス業：平均70%（原価率30%）
- わからない場合は業種平均値を入力

### P5-10: 売上増加の根拠 ⚠️ 重要

```
・予約率：15% = 月22件 → 保守的に月20件で計算
・客単価：5,000円
・月間売上：20件 × 5,000円 = 10万円（年間120万円）

【テイクアウト売上】
・平日：1日5組 × 単価3,000円 = 15,000円
・週末：1日10組 × 単価3,000円 = 30,000円
・月間：(15,000円 × 20日) + (30,000円 × 8日) = 54万円
・保守的に月20万円で計算（年間240万円）

【Instagram広告】
・広告費20万円 ÷ 3ヶ月 = 月6.7万円
・CPC（クリック単価）：50円 → 月1,340クリック
・来店率：1% = 月13人 → 保守的に月10人で計算
・客単価：5,000円
・月間売上：10人 × 5,000円 = 5万円（年間60万円）
```

# Phase 1質問順序（申請書準拠版）

## 設計方針

**申請書の記載順序に完全準拠した質問フロー**

様式2の構成：
```
【経営計画】
├─ 1. 企業概要
│   ├─ (1) 事業の概要
│   ├─ (2) 事業別売上高・利益の推移
│   ├─ (3) 立地と商圏
│   └─ (4) 業務状況と課題
├─ 2. 顧客ニーズと市場の動向
├─ 3. 自社の強み
├─ 4. 経営方針・目標と今後のプラン
└─ 5. 事業承継の計画（該当者のみ）

【補助事業計画】
├─ 1. 補助事業の内容
├─ 2. 経費明細表
└─ 3. 資金調達方法
```

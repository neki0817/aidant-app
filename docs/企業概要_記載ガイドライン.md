# 企業概要 記載ガイドライン

## 概要
小規模事業者持続化補助金の申請書類「経営計画書兼補助事業計画書(様式2)」における**1.企業概要**の記載方法をまとめたガイドラインです。

---

## 📋 記載すべき項目一覧

### 【1. 自社の概要】

#### ■ 基本情報
- **創業年・法人化した年**
- **業種・主要事業内容**
- **事業の特徴・コンセプト**
- **顧客層(個人/法人、年齢層、職業等)**

#### ■ 事業構造の表形式記載
売上を事業別・商品別に表で整理する

```
| 事業/商品 | 売上高 | 売上総利益 | 売上シェア | 客単価 |
|----------|--------|-----------|-----------|--------|
| A事業    | ○○千円 | ○○千円    | ○○%      | ○○円   |
| B事業    | ○○千円 | ○○千円    | ○○%      | ○○円   |
| 合計     | ○○千円 | ○○千円    | 100%      | -      |
```

**記載のポイント:**
- 売上高は千円単位で記載
- 売上総利益を明記することで収益性を示す
- 客単価で顧客あたりの価値を示す

---

### 【2. 立地場所】

#### ■ 記載すべき内容
- **アクセス方法**(最寄り駅からの距離、車でのアクセス等)
- **立地の特性**(商業地・住宅街・郊外等)
- **駐車場の有無**
- **立地のメリット・デメリット**

#### ■ 顧客動向
- **開業当初の顧客層**
- **現在の顧客層の変化**
- **地域外からの来店割合**
- **口コミ・SNS等での評判**

**記載例:**
```
当店は、○○駅から徒歩3分の商業ビル内に位置しています。
駅直結のため天候に左右されず、仕事帰りやショッピングの
ついでに立ち寄りやすい立地です。

開業当初は近隣在住の顧客が中心でしたが、SNSでの評判が
広がり、現在では近隣市からの来店も約30%を占めています。
```

---

### 【3. 主な商品・サービス】

#### ■ 商品・サービス別の詳細記載

各商品・サービスについて以下を記載:
- **商品名・サービス名**
- **価格(税込)**
- **営業利益率**
- **提供内容の詳細**
- **所要時間**(サービス業の場合)

**記載例(飲食業):**
```
### ランチメニュー
- Aコース(1,800円) 営業利益率 約30%
  - 前菜・メイン・デザート・コーヒー
- Bコース(2,500円) 営業利益率 約35%
  - 前菜2品・メイン・デザート・コーヒー
```

**記載例(サービス業):**
```
### ジェルネイルメニュー
- シンプルコース(6,000円) 営業利益率 約72%
  - ワンカラー・グラデーション・フレンチから選択
  - 所要時間: 60分
```

---

### 【4. 各商品の売上状況】

#### ■ 売上総額ランキング表

```
| 順位 | 商品名 | 年間売上 | 構成比 |
|------|--------|----------|--------|
| 1位  | ○○    | ○○千円   | ○○%   |
| 2位  | ○○    | ○○千円   | ○○%   |
| 3位  | ○○    | ○○千円   | ○○%   |
```

#### ■ 利益総額ランキング表

```
| 順位 | 商品名 | 年間利益 | 構成比 |
|------|--------|----------|--------|
| 1位  | ○○    | ○○千円   | ○○%   |
| 2位  | ○○    | ○○千円   | ○○%   |
| 3位  | ○○    | ○○千円   | ○○%   |
```

**記載のポイント:**
- 売上が多い商品と利益が多い商品は異なる場合がある
- 両方を記載することで事業構造が明確になる

---

### 【5. 売り上げの状況】

#### ■ 日次・月次の売上データ

**飲食業・サービス業の場合:**
```
【平日】
- ランチ: ○○組(平均○○名)
- ディナー: ○○組(平均○○名)

【週末】
- ランチ: ○○組(平均○○名)
- ディナー: ○○組(平均○○名)

【月間平均売上】: 約○○千円
【繁忙期(○月・○月)】: 月間売上○○千円超
```

**サロン・施術系サービス業の場合:**
```
【平日】
- 1日平均: ○～○名(予約枠○枠)
- 稼働率: 約○○%

【週末】
- 1日平均: ○～○名(予約枠○枠)
- 稼働率: 約○○%

【月間平均売上】: 約○○千円
【リピーター比率】: ○○%
【新規顧客】: 月平均○名
```

---

### 【6. 業務状況】

#### ■ 人員体制
- **従業員数**(正社員・アルバイト別)
- **役割分担**(製造・販売・管理等)
- **勤務シフト**

#### ■ 業務内容
個人経営の場合は担当業務を列挙:
- 施術業務・接客業務
- 予約管理・顧客管理
- SNS運用
- 仕入れ・在庫管理
- 経理業務

#### ■ 現在の課題

**必ず記載すべき課題:**
1. **人材確保の課題**
2. **業務効率化の課題**
3. **資金繰りの課題**

**記載例:**
```
【課題】
1. 人材確保の難しさ:
   調理スタッフの確保が困難で、シェフ1名に
   調理業務が集中している状況です

2. 予約管理の非効率性:
   電話とメモによる予約管理のため、ダブルブッキングや
   予約漏れのリスクがあります

3. 資金繰り:
   2021年のコロナ禍で売上が大幅減少し、○○銀行から
   3,000千円の借入を行い、毎月125千円を返済中です
   (残債:1,500千円、完済予定:2026年3月)
```

---

### 【7. 直近3年間の業績推移】

#### ■ 業績推移表

```
| 年度 | 売上高 | 売上総利益 | 営業利益 | 備考 |
|------|--------|-----------|----------|------|
| 2021年 | ○○千円 | ○○千円 | ○○千円 | コロナ禍で減収 |
| 2022年 | ○○千円 | ○○千円 | ○○千円 | 回復基調 |
| 2023年 | ○○千円 | ○○千円 | ○○千円 | コロナ前を回復 |
```

#### ■ 増減要因の記載

**売上増加要因の例:**
- Instagram投稿強化により認知度向上
- 新メニュー導入で客単価アップ
- リピーター向けポイントカード導入

**売上減少要因の例:**
- 新型コロナウイルスの影響
- 原材料高騰による価格転嫁
- 競合店の出店

---

## 🎯 業種別の記載ポイント

### 【飲食業】
- **メニュー別の客単価・営業利益率**を明記
- **ランチ/ディナー別**の売上構成
- **季節変動**(歳末・記念日需要等)
- **主要顧客層**(個人/法人、年齢層)

### 【サービス業(サロン・施術系)】
- **施術時間と回転率**
- **予約枠の稼働率**
- **リピーター率**の明記
- **SNS集客の状況**

### 【小売業】
- **商品カテゴリー別**の売上構成
- **客単価・購買頻度**
- **在庫回転率**
- **仕入先との関係**

### 【製造業】
- **製造工程**(内製化率)
- **生産能力**(月間生産量等)
- **OEM/自社ブランド**の比率
- **技術的な強み**

---

## 📊 数値データの記載ルール

### 【金額表記】
- **千円単位**で記載(例: 8,000千円)
- 税込/税抜を明記

### 【比率表記】
- **パーセント表記**で記載(例: 75%)
- 小数点第1位まで

### 【期間表記】
- **年月を明記**(例: 2023年12月期)
- 直近3年間のデータを掲載

## 📝 自動生成時の注意点

アプリで企業概要を自動生成する際の注意点:

1. **数値の一貫性**: 売上合計と内訳の合計が一致するか確認
2. **時制の統一**: 「現在」「直近」などの時制を統一
3. **業種特性の反映**: 業種ごとに重要な指標が異なる
4. **具体性**: 抽象的な表現を避け、数値で示す
5. **課題の明確化**: 補助事業で解決したい課題を明記

---

## 🔄 他セクションとの関連

企業概要で記載した内容は、以下のセクションでも活用される:

- **2. 顧客ニーズと市場の動向** → 顧客層の詳細分析
- **3. 自社の強み** → 商品・サービスの差別化要素
- **4. 経営方針・目標** → 課題解決のための具体策

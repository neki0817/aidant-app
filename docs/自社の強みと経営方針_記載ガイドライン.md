# 自社の強みと経営方針 記載ガイドライン

## 概要
小規模事業者持続化補助金の申請書類「経営計画書兼補助事業計画書(様式2)」における**3.自社の強み・弱み**と**4.経営方針・目標と今後のプラン**の記載方法をまとめたガイドラインです。

---

## 📋 【3. 自社の強み・弱み】記載項目

### 【強みの記載方法】

#### ■ 基本方針
- **具体的なエピソード・数値**で示す
- **顧客からの評価**を根拠にする
- **競合との差別化要素**を明確にする
- **業種特性**に合わせた強みを記載

### 【弱みの記載方法】

#### ■ 基本方針
- **正直に課題を認める**（隠さない）
- **具体的な対応策を示す**（解決意欲を見せる）
- **補助事業での解決を示唆する**

#### ■ よくある弱み・課題

**1. 認知度の低さ**
```
### 地域での認知度不足
開業○年目で、地元○○市内での認知度は高まってきましたが、
隣接市町村での認知度はまだ低い状況です。

対応策:
- 地域誌への広告出稿（本補助事業で実施予定）
- SNS広告の強化
- チラシ配布による認知度向上
```

**2. 人手不足**
```
### スタッフの確保が困難
現在○名体制で運営していますが、繁忙期には
人手が不足し、お客様をお待たせすることがあります。

対応策:
- 業務効率化による生産性向上（本補助事業で予約システム導入）
- パート・アルバイトの募集強化
- スタッフの多能工化
```

**3. 設備の老朽化**
```
### 設備の老朽化
○○年前に導入した○○が老朽化しており、
故障のリスクが高まっています。

対応策:
- 新設備の導入を検討（本補助事業または自己資金）
- メンテナンスの頻度を上げる
```

## 📋 【4. 経営方針・目標と今後のプラン】記載項目

### 【経営方針の記載方法】

#### ■ 基本方針
- **一文で表現できるコンセプト**を明記
- **3～5つの柱**で具体化
- **顧客目線**での記載

**記載例:**
```
## ■ 経営方針

当店は、「○○○○」というコンセプトのもと、
地域の皆様に愛される店づくりを目指しています。

### 3つの柱
1. ○○へのこだわり: ○○○○
2. ○○への対応力: ○○○○
3. ○○な雰囲気: ○○○○
```

---

### 【経営目標の記載方法】

#### ■ 短期目標（1年後）

**必須記載事項:**
- 売上目標（金額・前年比）
- 営業利益目標（金額・前年比）
- その他の数値目標（新規顧客数・リピーター率等）

**記載例:**
```
### 【短期目標】2024年12月期

#### 売上目標: 28,000千円（前年比+12%）

内訳:
- ○○事業: ○○千円（前年比+○%）
- ○○事業: ○○千円（前年比+○%）

#### 営業利益目標: 2,800千円（前年比+16.7%）

原価率の改善とオペレーション効率化により、
売上増加率を上回る利益成長を目指します。

#### 新規顧客目標: 年間300名

特に○○地域からの新規顧客獲得に注力します。
```

---

#### ■ 中期目標（3～5年後）

**記載内容:**
- 売上目標
- 事業拡大計画（2店舗目出店、新事業等）
- 質的な目標（ブランド確立、地域No.1等）

**記載例:**
```
### 【中期目標】2026年12月期

#### 売上目標: 35,000千円

- 2店舗目の出店を視野に入れた成長戦略
- ○○事業の分社化による効率化

#### リピーター率: 75%以上維持

新規顧客獲得と並行して、既存顧客の満足度維持を重視します。
```

### 【今後のプランの記載方法】

#### ■ 基本方針
- **時系列**で整理（いつ・何をするか）
- **補助事業の内容**を明確に記載
- **期待効果**を数値で示す

#### ■ 記載フォーマット

```
### 【20○○年○～○月】○○（本補助事業）

#### ○○の実施
○○を行います。

- 実施時期: ○月
- ターゲット: ○○
- 内容: ○○

期待効果:
- ○○: ○○
- 売上増加: 月間+○○千円
```

**記載例（飲食業）:**
```
### 【2024年4～6月】販路開拓（本補助事業）

#### 地域誌への広告出稿
隣接A市・B町で配布されている地域誌「○○マガジン」に
広告を掲載します。

- 掲載時期: 5月号・6月号（各1ページ）
- ターゲット: 40～60代の富裕層
- 訴求内容: 記念日コース、個室完備

期待効果:
- A市・B町からの新規顧客: 月間10名増加
- 売上増加: 月間+100千円（客単価10,000円×10名）

#### チラシのポスティング
A市・B町の住宅街に、3,000枚のチラシをポスティングします。

- 実施時期: 6月上旬
- 内容: メニュー紹介、初回来店クーポン付き

期待効果:
- 反応率1%と仮定: 30組の新規来店
- 売上増加: 約250千円
```

**記載例（サロン業）:**
```
### 【2024年5～7月】新規顧客獲得（本補助事業）

#### Instagram広告の実施
Instagram広告を3ヶ月間実施し、○○市・A市在住の
20～40代女性をターゲットに認知度向上を図ります。

- 広告予算: 月間50,000円×3ヶ月
- リーチ目標: 延べ30,000名
- 訴求内容: ビフォーアフター写真、初回割引クーポン

期待効果:
- 新規顧客: 月間15名増加
- 売上増加: 月間+90千円（客単価6,000円×15名）
```

## 🤖 AI自律エージェントによる生成方針

### 【強みの生成ロジック】

#### 1. **Google Maps口コミ分析**
- 高評価レビューから頻出キーワード抽出
- 評価点数の集計
- レビュー件数の集計

#### 2. **業種別の強みテンプレート適用**
- 飲食業: 味・接客・雰囲気・価格
- サロン業: 技術力・マンツーマン・衛生管理
- 小売業: 品揃え・接客・立地
- 製造業: 技術力・納期・カスタマイズ

#### 3. **競合比較データの生成**
- Google Maps検索で同業種店舗を抽出
- 価格帯・評価点数を比較
- 差別化ポイントを明確化

---

### 【弱みの生成ロジック】

#### 1. **口コミから改善要望を抽出**
- 低評価レビューからネガティブキーワード抽出
- 改善要望の集計

#### 2. **業種別の典型的な課題を推定**
- 飲食業: 駐車場不足・予約の取りにくさ
- サロン業: 予約枠の少なさ・認知度不足
- 小売業: 在庫不足・駐車場問題
- 製造業: 設備老朽化・人手不足

#### 3. **対応策の自動生成**
- 補助事業で解決できる課題は「本補助事業で実施」
- それ以外は一般的な対応策を提示

---

### 【経営目標の生成ロジック】

#### 1. **現在の売上から目標を算出**
- 短期目標: 現在売上の110～120%
- 中期目標: 現在売上の140～150%

#### 2. **補助事業の効果を反映**
- 広告出稿による新規顧客増
- 設備導入による生産性向上
- 効率化による利益率改善

---

### 【プランの生成ロジック】

#### 1. **補助事業の内容を時系列で整理**
- ユーザーが選択した補助金の使途（Q1-9の回答）
- 業種に応じた標準的な実施スケジュール

#### 2. **期待効果の数値化**
- 業種別の標準的な効果率を適用
- 広告: 反応率1～3%
- 設備: 生産性10～30%向上
